# Guthrie Corning Hospital — ROI Form Retrieval Notes

## Search Strategy

1. **Web search** for `Guthrie Corning Hospital "authorization" "release" "health information" form PDF` — immediately found the main ROI form PDF hosted on guthrie.org.
2. **Web search** for `Guthrie Corning Hospital "medical records" "release form" PDF` — confirmed the same form and found the authorization instructions PDF.
3. **Web search** for `site:guthrie.org authorization release medical records form` — found additional Family Access release form.

All forms are published by the parent system (Guthrie Clinic / guthrie.org) and apply to all Guthrie facilities including Corning Hospital. The medical records page at https://www.guthrie.org/medical-records serves as the central hub.

## Difficulties

- **None.** Forms were easy to find via web search and directly downloadable as PDFs without any bot blocking or JavaScript barriers.
